BEM Metodologija

BEM je kratica za Block, Element, Modifier

BEM omogućava prilično strog način organizacije CSS klasa u neovisne module. To je objektan način pisanja CSS klasa i izgleda otprilike ovako:

.block {}

.block__element {}

.block--modifier {}

.block__element--modifier {}

Blok predstavlja objekt na web stranicama, npr menu
Element je dio bloka, npr menu__item - oznaka su dva underscora

Modifier je varijacija elementa ili bloka, npr menu__item--news (ukoliko postoji razlika u recimo boji rubrika - news (crveno), show (narančasto), sport (zeleno)) - oznaka su dva minusa --
